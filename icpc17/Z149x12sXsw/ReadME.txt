The data folder ICPC17 includes 12 subfolder. Each subfolder include the data for one video. 

1- For each video we provide the following information

  a- all_comments.txt: includes all the comments sampled from the video’s list of comments
  b- concerns.txt: includes all the comments that were manually classified as content concerns
  c- other.txt: includes all the comments that were manually classified as miscellaneous
  d- user_sum.txt*: includes the list of our experiment participants’ comment summaries. Participants summaries are separated by "*****" 

*Note that this data is only available for the videos used in the human experiment (summarization phase)
